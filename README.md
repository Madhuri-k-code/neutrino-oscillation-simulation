# Neutrino Oscillation Simulation

This project demonstrates a simple simulation of neutrino oscillations using Python.

## Overview
Neutrinos can change their flavor as they propagate. This simulation models the oscillation probability as a function of distance.

## Features
- Basic two-flavor oscillation model
- Visualization using matplotlib
- Simple and extendable structure

## Future Work
- Include decoherence effects
- Extend to three-flavor oscillations
- Improve numerical modeling

## Tools Used
- Python
- NumPy
- Matplotlib
  
## Decoherence Extension

This project now includes a simple model of decoherence, where an exponential damping factor is applied to the oscillation probability.

This demonstrates how quantum coherence can gradually disappear due to environmental interactions.
